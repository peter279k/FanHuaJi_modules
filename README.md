# 2D-Gate 繁化姬 詞語模組

2D-Gate 繁化姬：https://sctctw.2d-gate.org

這個倉庫是「2D-Gate 繁化姬」中所使用的模組，你可以提交新的模組或是修正現有的模組。


## 環境要求

[PHP](http://php.net) ≥ 5.6
